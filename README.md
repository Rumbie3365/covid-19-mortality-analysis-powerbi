# covid-19-mortality-analysis-powerbi
Health data analysis project using Excel, SQL and Power BI to analyse COVID-19 cases, deaths and mortality trends.

## Project Overview

This project analyses COVID-19 cases and deaths using publicly available WHO COVID-19 data. The purpose of the project is to identify mortality trends, compare countries and regions, calculate case fatality rates, and present the findings through an interactive Power BI dashboard.

The project demonstrates practical skills in Excel data preparation, SQL analysis, Power BI dashboard development, health data interpretation, and public health reporting.

## Tools Used

- Microsoft Excel
- SQL
- Power BI
- WHO COVID-19 global dataset

## Key Questions

1. What were the global COVID-19 case and death totals?
2. Which countries recorded the highest reported COVID-19 deaths?
3. How did COVID-19 deaths change over time?
4. Which countries had the highest case fatality rates?
5. How did Zimbabwe’s COVID-19 mortality trend change over time?
6. What are the main limitations when interpreting reported COVID-19 data?

## Key Insights

- COVID-19 mortality varied significantly between countries and regions.
- Total deaths alone do not provide a complete comparison because population size differs between countries.
- Case fatality rate gives additional context, but it must be interpreted carefully because it is affected by testing levels, healthcare access and reporting quality.
- Zimbabwe recorded lower total deaths compared with many larger countries, but the trend still shows clear periods of increased mortality.
- Weekly death trends can help public health teams identify periods of increased disease burden and possible pressure on health systems.

## Data Limitations

- Reported COVID-19 deaths may not capture all actual deaths.
- Reporting methods differ between countries, which affects comparability.
- Some countries may have underreported cases or deaths due to limited testing, delayed reporting or changes in public health reporting rules.
- Case fatality rate is affected by testing coverage, age structure, healthcare access and reporting completeness.
- The dashboard is designed for trend analysis and reporting, not as a full epidemiological model.
